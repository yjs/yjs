# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 13.6.x   | :white_check_mark: |
| < 13.6.0   | :x:                |

## Reporting a Vulnerability

Open an issue. If the vulnerability can be used to exploit user data (or leak other sensitive information) send an email to yjs [at] kevinjahns.de
